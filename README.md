# Simulación de seis grados de separación

Este proyecto tiene como objetivo general verificar empíricamente la teoría de los seis grados de separación mediante una simulación en Python, utilizando librerías para poder realizar afirmaciones basadas en datos.

## Ejecución

Para ejecutar la simulación se debe ejecutar el siguiente comando:

```bash
    uv run python main.py
```

Es necesario tener instalado el gestor de paquetes [uv](https://github.com/astral-sh/uv)

## Librerías

Para este proyecto, se utilizaron las siguientes librerías:

- **NetworkX**
- **NumPy**
- **Matplotlib**
- **Pandas**
